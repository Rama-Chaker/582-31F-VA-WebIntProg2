Reflection Answers:

  1. The purpose of type="module" in the script tag is to enable the use of JavaScript modules, allowing us to import and export functionality between different files.
  2. It is useful to separate fetchUsers() in a api.js file to keep the code and logic organized and maintainable.
  3. It is useful to separate the UI rendering logic into a ui.js file to keep the code organized and maintainable.
  4. It will be hard to debug if we keep everything in one place.
  5. The difference between exporting and importing : exporting allows us to make certain variables or functions available to other modules, while importing allows us to use those exported variables or functions in our current module.
